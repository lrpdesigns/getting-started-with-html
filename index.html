The Basics of the Web & HTML

Major Pieces of the Web

The web is a ton of computers that communicate with each other. When you go to a web page your computer sends a request to a server. The server finds the requested HTML document and sends it back to your computer, that then interprets the page and displays it on your screen.

HTML - Hypertext Markup Language

HTML documents are the most common form of content on the web. HTML documents are made up of:

• text content - what you see
• markup - how it looks
• reference to other docs - e.g. images and videos
• links to other pages

HTML ATTRIBUTES/ELEMENTS

HTML documents are made of HTML attributes and elements. When writing HTML, you can tell the browsers the type of element by using different "tags".

TYPES OF TAGS:
• Anchors - (A) Used for placing a link on your page.
• Images - (IMG) Used to place an image or photo onto your page.
• Whitespce - (BR) Used to create a break in any of your type. Another form of whitespace is (P) used to create paragraphs.

Inline and Block Elements

HTML elements are either inline or block. Block elements have a sort of invisible box around the content inside of them allowing for more flexibility with the content they hold.
